==============WSL==============
WSL (Windows Subsystem for Linux) es una característica de Windows 10/11 y Windows Server que permite ejecutar un entorno 
GNU/Linux nativo directamente en Windows sin la sobrecarga de una máquina virtual o el arranque dual

Pasos para instalar: 
1) wsl --install        instala la aplicacion 
2) wsl --version        muestra version de wsl
3) wsl --list --online   muestra versiones de Linux disponibles
4) wsl --install Ubuntu-24.04
5) wsl -l -v            muestra distribuciones instaladas
6) wsl -d <Distro>      Ingresar a distribución

Integracion con dockert desktop:
1) ir a docker desktop Settings->general->Use the WSL 2 based engine.
1) ir a docker desktop Settings->Resources->WSL Integration->Enable integration with additional distros.
3) seleccionar la version de linux que se esta usando con WSL.

=============COMANDOS=========
-listar: wsl -l -v
-Detener proceso especifico: wsl --terminate Ubuntu-24.04
-Detener todo: wsl --shutdown
-Levantar todo: wsl 
-Levantar especifico: wsl -d Ubuntu-24.04
